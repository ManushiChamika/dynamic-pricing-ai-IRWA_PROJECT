## Technical Implementation Details

**Progress Animation Logic:**
- Initial state: uploadProgress = 0
- Interval increments: +10 every 150ms
- Cap logic: Math.min(prev + 10, 90) prevents early 100%
- On completion: clearInterval + setUploadProgress(100)
- Reset: setTimeout with 1000ms delay for visual polish

**Theme-Aware Styling Pattern:**
- const isDark = theme === 'dark' at component top
- Ternary operators for conditional classes
- Consistent patterns: isDark ? 'bg-gray-800' : 'bg-white'
- Maintains accessibility in both modes
- Color palette specifically chosen for readability

**File Size Formatting Algorithm:**
- Bytes < 1024 ‚Üí display as B
- Bytes < 1MB ‚Üí divide by 1024, 1 decimal (KB)
- Default ‚Üí divide by 1024¬≤,  2 decimals (MB)
- Example conversions accurate and user-friendly
