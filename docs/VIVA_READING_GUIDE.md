## ðŸ“– READING STRATEGIES

### Strategy 1: "Skim First"
1. Read section headings only
2. Read first paragraph of each section
3. Read bullet points
4. On second pass, read full text

**Best for:** Long documents, tight schedule

---

### Strategy 2: "Read with Purpose"
1. Before reading, write 3 questions you want answered
2. Read looking for answers to those questions
3. Highlight answers when you find them
4. Summarize answers in 1-2 sentences

**Best for:** VIVA_COMMUNICATION_PROTOCOL_GUIDE.md

---

### Strategy 3: "Active Reading"
1. Read paragraph
2. Close document
3. Write down what you remember
4. Re-read to check accuracy
5. Note anything you forgot

**Best for:** VIVA_QUICK_REFERENCE.md (for memorization)

---

### Strategy 4: "Visual Learning"
1. Read PRESENTATION_VISUAL_1PAGE.md first
2. Look at diagrams and ASCII art
3. Then read text documents
4. Connect concepts to visual diagrams

**Best for:** Understanding system architecture
